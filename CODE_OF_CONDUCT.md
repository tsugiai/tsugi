# Code of Conduct

This project adopts the Contributor Covenant Code of Conduct version 2.1 by
reference:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

We pledge to make participation in this project respectful, harassment-free, and
welcoming to contributors with different backgrounds, experience levels, and
perspectives. Contributors are expected to engage constructively, assume good
faith, and keep discussion focused on improving the project.

To report a code-of-conduct concern, contact `tong@tsugicinema.com`. Maintainers
will review reports privately and take proportionate action consistent with the
Contributor Covenant v2.1 enforcement guidance.
